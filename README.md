Customer Churn Analysis System (Synthetic Data)
Overview
This project analyzes customer churn using synthetic data. It generates realistic customer records and studies patterns to understand why customers leave a service. The goal is to help businesses improve customer retention.

Objectives
Specific Objectives
Generate synthetic customer data
Perform data cleaning and preprocessing
Analyze customer behavior
Identify churn patterns
Create visualizations
Overall Objective
To identify key factors influencing customer churn and provide useful insights.

Project Highlights
Generated 1,00,000 customer records
Cleaned and processed data
Identified churn patterns
Performed EDA
Created visualizations
Derived insights
Data Preprocessing
Checked missing values
Verified data types
Converted categorical data (Gender) to numeric
Created AgeGroup feature
Ensured data consistency
Exploratory Data Analysis (EDA)
Calculated mean, median, standard deviation
Compared churn vs non-churn customers
Analyzed age, gender, activity, and product usage
Identified trends and patterns
Machine Learning Model
Logistic Regression or Decision Tree can be used
Helps predict customer churn
Visualization
Bar chart: Churn distribution
Histogram: Age distribution
Scatter plot: Balance vs Credit Score
Box plot: Balance vs Churn
Heatmap: Correlation

Conclusion
Customers with low credit score, high balance, and inactive status are more likely to churn.
